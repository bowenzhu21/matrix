# Matrix

Matrix is an artificial society of graph-node LLM agents modeled on human demographic and behavior patterns using Exa, propagating via weighted BFS scored by social proximity and societal influence, with Supermemory for context.

If you want a live demo, contact me at bowenzhu66@gmail.com and I'd be happy to demo it for you.
